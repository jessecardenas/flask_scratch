# flask_scratch

Basic boilerplate and template to get flask app running

## to set up
 python3 -m venv venv  
 . venv/bin/activate  
 pip install -r requirements.txt  

## to run
 venv/bin/python3 app.py  

## features:
 test script
 config  
 basic flask app
